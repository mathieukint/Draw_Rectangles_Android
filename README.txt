+-----------------------------------------------+
|	Application de dessin de rectangle	|
+-----------------------------------------------+

Application mobile qui permet de dessiner des 
rectangles selon les dimensions spécifiées :
 - Largeur
 - Hauteur

Il est également possible de choisir l'emplacement 
du rectangle à dessiner.

La combinaison des deux paramètres permet de 
dessiner plusieurs rectangles de différentes 
tailles et à différents endroits de l'écran, 
ainsi que de les superposer.



Emulation optimale sur Pixel_3a_API_30_x86
